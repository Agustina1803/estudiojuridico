# Estudio Juridico APJD ⚖️

Aplicación web responsiva diseñada para optimizar la organización interna de un estudio jurídico. Permite el acceso diferenciado por rol (Secretario, Abogado, Administrador), con funcionalidades específicas para cada perfil: gestión de casos, carga de documentación, control de citas, control de usuarios.

## Demo del proyecto
Mira el demo del proyecto
[aqui](https://apjsestudiojuridico.netlify.app/)
